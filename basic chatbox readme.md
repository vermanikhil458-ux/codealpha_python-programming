Basic Chatbot
Description

This is a simple rule-based chatbot built using Python. It responds to predefined user inputs such as greetings, questions, and farewells.

Features
Responds to greetings like "hello", "hi", and "hlo"
Answers simple questions
Uses a loop for continuous conversation
Exits when the user types "bye"
Technologies Used
Python
if-elif statements
Functions
Loops
Input/Output
How to Run
Install Python on your computer or directly open in online compiler
Save the chatbot code in a file named chatbot.py.
Open a terminal or command prompt.
Run the program:
python chatbot.py
Example
ChatBot: Hello! Type 'bye' to exit.

You: hi
ChatBot: Hi!

You: how are you
ChatBot: I'm fine, thanks!

You: bye
ChatBot: Goodbye!
